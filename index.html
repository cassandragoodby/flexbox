How to Build a Website

1

Learn HTML

HTML is hypertext markup language and is the coding language used for text on the internet. It is the first step to creating a website. Every wondered why webpages end in .html?

2
Learn CSS

Next is to learn CSS or cascading style sheets. This is the core of the design that works along with the HTML document.

3
Learn Javascript

Javascript is an object-oriented computer programming language commonly used to create interactive effects within web browsers.

4
Cry a lot!

Web designing takes a lot of practice and time! Everything in your code will break and you will have late nights.
